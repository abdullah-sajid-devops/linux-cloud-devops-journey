# Day 26 of Cisco Linux Essentials Course

Today I learned about GUI (Graphical User Interface) and the desktop environment basics, along with several other important topics.

## 1. Linux's Desktop Ecosystem
Just like Windows has its own desktop and application software, Linux desktop environments also have their own set of applications available.

## 2. Email Desktop Clients
- **Mozilla Thunderbird** — the most famous desktop client, made by the Mozilla Foundation, fully-featured
- Other clients: **Evolution**, **KMail**
- Terminal-based mail tools: **mutt**, **alpine**, **mailx**

## 3. Shell
The shell is what takes commands from us and gets them executed via the kernel. It provides a rich scripting language for automating tasks.

**Shell families:**
1. Bourne shell
2. C shell

**Modern shells:**
1. Bash
2. tcsh
3. ksh
4. Zsh

## 4. Text Editors
Common Linux text editors: vi/vim, Emacs, Nano, Pico.
- Nano and Pico — simple, easy to use, low learning curve
- Vi and Emacs — most advanced, plugin/feature-rich, but harder for beginners. Still considered essential knowledge for sysadmins.

## 5. Creative Tools
Linux has excellent creative tools similar to Windows, including Blender, GIMP, Audacity, and more.

## 6. Office Suites
Word processors, spreadsheets, and presentation tools together make up an office suite. Microsoft Office is most common on Windows.

**LibreOffice vs OpenOffice:** Both are complete office suites — LibreOffice is a fork of OpenOffice.

**Main LibreOffice apps:**
1. LibreOffice Calc (Excel alternative)
2. LibreOffice Writer (Word alternative)

## 7. Web Browsers
Mozilla Firefox and Google Chrome both run smoothly on Linux, offering a very similar experience to Windows.
